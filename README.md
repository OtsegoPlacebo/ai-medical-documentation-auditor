
# AI Medical Documentation Auditor — Version 2.0

Created by Janyce Viero

## Project Overview

This is a polished Health Information Management and AI portfolio project. It demonstrates how an AI-assisted documentation review tool can help identify:

- Missing documentation elements
- Coding support concerns
- Compliance concerns
- Provider query opportunities
- Documentation quality score
- Risk level

## Important Disclaimer

This app is for educational and portfolio use only. It does not provide final medical coding, clinical, billing, legal, or compliance advice.

## Files

- `streamlit_app.py` — main web app
- `requirements.txt` — libraries needed for deployment
- `README.md` — project instructions

## How to Run Locally

Install requirements:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run streamlit_app.py
```

## How to Deploy on Streamlit Community Cloud

1. Create a GitHub account.
2. Create a new repository.
3. Upload these files:
   - `streamlit_app.py`
   - `requirements.txt`
   - `README.md`
4. Go to Streamlit Community Cloud.
5. Connect GitHub.
6. Select your repository.
7. Set the main file path as:

```text
streamlit_app.py
```

8. Click Deploy.

## Portfolio Description

Developed an AI-assisted healthcare documentation review app using Python and Streamlit. The tool evaluates clinical documentation for completeness, coding support, compliance concerns, and provider query opportunities. It includes a dashboard, sample cases, scoring rubric, and downloadable audit report.
